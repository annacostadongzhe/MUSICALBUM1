# 尺八冥想音乐 · 项目策划书

## 项目概述

**作品名称**：暂定《虚竹》（Empty Bamboo）—— 尺八冥想音乐作品

**核心概念**：以尺八（Shakuhachi）为主体乐器，融合人声（诵念、气息、泛音唱法）、自然环境音（流水、风、竹林、鸟鸣、溪石），营造一片可沉浸的"声音禅境"。整首作品不是"演奏"给听众听，而是为听众搭建一个可以"住进去"的声场。

**时长建议**：10–20 分钟（完整冥想版）/ 3–5 分钟（精简引导版）

## 一、声音素材规划

### A. 尺八（核心声源）
| 素材 | 说明 |
|------|------|
| 传统本曲音阶（Koku Reibo、Koro Koro 等） | 低音悠长、气息表现力强 |
| 气息音（吹口气流声） | 制造"呼吸即音乐"的效果 |
| 指滑音、半音 | 增加不确定性和禅意 |
| 超高音（Overblow） | 间或使用，制造空灵感 |
| 击板声（琴体敲击） | 作为节奏锚点或过渡 |

### B. 人声
| 素材 | 说明 |
|------|------|
| 低沉诵念（Om / 阿字观） | 低频基底，与尺八形成泛音共鸣 |
| 口呼吸声（吸入/呼出） | 与尺八气息音呼应，形成"呼吸对话" |
| 泛音唱法（Overtone Singing） | 点缀性使用，增加空间层次感 |
| 无词的元音延长（Ah / Ohm / Hum） | 长音铺底，营造包围感 |
| 耳语 / 微声吟诵 | 近距离感，制造亲密冥想氛围 |

### C. 自然声景
| 素材 | 说明 |
|------|------|
| 流水（溪流、泉水滴落） | 节奏天然多变，可以做全曲的背景脉冲 |
| 风穿过竹林 | 高、中频白噪声，与尺八高频噪声融合 |
| 鸟鸣（夜间 / 清晨） | 点缀性出现，标记段落转换 |
| 木鱼 / 钟 / 磬 | 极少量使用，作为"时间锚点" |
| 脚步声（落叶/木板） | 若做成"行走冥想"则可作为引导线索 |

## 二、曲式结构建议

### 方案 A：线性沉浸版（推荐）

`
[ 0:00 – 2:00 ]  入境
  声场：远处溪流 + 微微风声（极低音量）
  尺八：气息音为主，偶尔出现一个音符，拉长 20-30 秒
  人声：遥远的 Ah 泛音，若隐若现

[ 2:00 – 5:00 ]  呼吸
  声场：溪流渐近，水滴声出现
  尺八：引入简单动机（3–5 音），大量留白
  人声：引入呼吸声（左/右声道交替），与尺八气息"对话"
  节奏感：几乎无节拍，跟着"呼吸周期"走

[ 5:00 – 9:00 ]  展开
  声场：风声渐强，竹林摇曳声
  尺八：完整乐句浮现，Muraiki（强吹）技法出现
  人声：低沉诵念加入（Om），整体氛围加深
  动态：音量缓慢爬升，情绪保持内敛不张扬

[ 9:00 – 13:00 ]  禅定
  声场：所有自然音趋于平稳，形成绵密的声底
  尺八：音符之间间隔拉大，力度减弱
  人声：耳语或无声之息
  感受：音乐几乎静止，但声场仍然活着

[ 13:00 – 15:00 ]  出定
  声场：自然音渐渐退远
  尺八：最后一次呼气般的音符，渐弱至无声
  人声：一声悠长的呼气
  结束：静默 10-15 秒后自然淡出
`

### 方案 B：循环冥想法（适合短循环播放/瑜伽/冥想课）

将 3-5 分钟的循环段设计成一个"呼吸周期"：

`
低音尺八 Drone（持续） + 人声 Ohm（持续）
→ 尺八 Melody（2-3 个音，周期出现）
→ 自然元素轮换（每周期换一种主自然声）
→ 结尾渐弱/直接循环无缝衔接
`

## 三、混音与空间设计

- **立体声宽度**：使用双耳录音/立体声扩展，自然声在 LCR 通道间缓慢漂移
- **纵深层次**：
  | 层次 | 内容 | 位置 |
  |------|------|------|
  | 前景 | 尺八 + 人声主音 | 中置，稍近距离 |
  | 中景 | 环境乐器（木鱼、磬） | L/R 稍远 |
  | 背景 | 溪流、风、竹林 | 远场，立体声全开 |
- **混响**：模拟竹林空间 / 山间寺庙内部卷积混响（IR），中长混响时间（2-4 秒）
- **动态控制**：避免压缩器过度使用；保留自然动态，仅做平滑音量包络

## 四、生产流程

`
Phase 1: 素材采集
  ├── 尺八录音（干声 + 气息 + 技巧素材）
  ├── 人声录音（诵念、呼吸、泛音）
  ├── 自然声素材（版权免费 / 实地录制）
  └── 选择并下载高质量 CC0 音效库

Phase 2: 编排与 AI 辅助生成
  ├── 用 AI 生成过渡性声音纹理（Drone、Pad）
  ├── 将录音素材切片编排为曲谱/时间线
  └── 使用 DAW（Reaper / Ableton / Logic）搭建雏形

Phase 3: 混音与母带
  ├── 空间布局（Pan + Reverb + Delay）
  ├── 自动化音量/滤波器包络
  ├── 沉浸式母带（Loudness 控制在 -16 ~ -20 LUFS）
  └── 导出高分辨率版本（48kHz/24bit WAV + 320k MP3）

Phase 4: 包装与应用
  ├── 封面视觉 / 冥想引导文案
  ├── 时长版本：完整版 / 短版 / 循环版
  └── 交付格式：WAV / FLAC / 流媒体优化
`
