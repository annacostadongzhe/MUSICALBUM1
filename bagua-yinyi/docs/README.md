# 八音演易 · Eight Sounds Manifesting the Yijing

## 概念 Concept

以《易经》六十四卦为骨架，用八种中国民族乐器分别代表八卦，
每一卦又与自然界对应的声音结合，构成 8 x 8 = 64 卦音阵。

## 八卦与八器 Trigram—Instrument Mapping

Can Qian / Heaven — Bianzhong Bells — Thunder
Dui Dui / Lake — Dizi Flute — Lake Waters
Li Li / Fire — Pipa — Fire Crackling
Zhen Zhen / Thunder — Drum — Thunderclap
Xun Xun / Wind — Xiao Flute — Wind Soughing
Kan Kan / Water — Guzheng — Water Flowing
Gen Gen / Mountain — Stone Chime — Mountain Echo
Kun Kun / Earth — Xun Ocarina — Earth Resonance

## 结构 Structure

64 tracks = 8 chapters x 8 tracks.
Each chapter uses one lower trigram as the base,
with upper trigrams changing chapter by chapter.

Row = lower trigram (base instrument + base natural sound)
Column = upper trigram (overlaid instrument + overlaid natural sound)
