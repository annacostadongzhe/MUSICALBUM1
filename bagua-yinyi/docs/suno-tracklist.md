# 八音演易 · 六十四卦音阵 Suno 生成方案

## 系统架构

64 tracks = 8 chapters x 8 tracks.
Each chapter uses one lower trigram as the base (instrument + natural sound),
with upper trigrams changing sequentially.

Lower trigram = base instrument + base natural sound (dominant atmosphere)
Upper trigram = overlaid instrument + overlaid natural sound (textural layer)
